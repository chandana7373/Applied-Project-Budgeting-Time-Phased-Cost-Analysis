# Project Financial Planning: Time-Phased Budget Model

## Overview
This project demonstrates a **time-phased project budget** developed using Microsoft Excel to illustrate how project costs are planned, distributed, and monitored across a defined project timeline. The model reflects practical project financial planning techniques commonly used by project managers to control costs and track cumulative spending.

The budget is structured to align project activities with their respective timelines, enabling clear visibility into **weekly cost allocation** and **overall budget performance**.

---

## Objective
The primary objective of this project is to:
- Apply **time-phased budgeting concepts** to a project scenario
- Allocate costs across project activities and timelines
- Track **cumulative project expenditure**
- Demonstrate financial planning and cost control principles used in real-world project management

---

## Key Features
- Time-phased allocation of project costs
- Activity-level budgeting aligned with the project schedule
- Weekly cost distribution for improved cost tracking
- Cumulative budget calculations to monitor total spending
- Clear and structured Excel-based financial model
